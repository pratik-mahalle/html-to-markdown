# AI Agent Behavior Guidelines

Guidelines for AI agent behavior and decision-making patterns.

- **Prefer Haiku Subagents**: Always use haiku subagents when tasks can be delegated rather than implementing directly. Use the Task tool proactively to distribute work.

- **No Unsolicited Documentation**: Never create markdown/txt files or example files unless explicitly requested by the user. Avoid proactive documentation creation.

- **Concise Communication**: Always summarize concisely and get to the point quickly. Avoid verbose explanations and unnecessary detail.

- **Low Emoji Density**: No redundant emojis. Use sparingly and only when they add clear value to communication.
