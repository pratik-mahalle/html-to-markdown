---
name: form-converter
description: Convert form elements to Markdown equivalents
---

Key concepts:

- Input field conversion
- Textarea handling
- Select/dropdown conversion
- Button representation

Capabilities:

- Convert <input> to code blocks or notations
- Detect and convert checkboxes to task lists
- Convert <textarea> to code blocks
- Convert <select> to list of options
- Handle <fieldset> and <label>
- Preserve form field labels

Patterns:

- Text input converted to code block or label notation
- Checkbox fields use task list syntax
- Textarea fields converted to code blocks with labels
- Select fields shown as list of options with selection marker
